📋 О проекте
ONE VALVE System Cleaner — это мощный и современный инструмент для очистки операционной системы Windows от цифрового мусора, оптимизации производительности и мониторинга дискового пространства. Программа сочетает в себе профессиональные возможности с интуитивно понятным интерфейсом, вдохновлённым лучшими дизайн-решениями популярных сервисов.

✨ Особенности
🗑️ Многоуровневая очистка — удаление временных файлов, кэша браузеров, системных логов

📁 Управление загрузками — автоматическая очистка папки Downloads от установочных файлов

🔄 Дедупликация файлов — поиск и удаление дубликатов на рабочем столе

📊 Мониторинг дисков — визуализация занятого и свободного пространства

🎨 Современный интерфейс — 6 уникальных тем оформления

🚀 Многопоточность — интерфейс не зависает во время выполнения задач

🎯 Возможности в деталях
🔍 Очистка временных файлов
Программа сканирует системные папки TEMP, TMP, AppData\Local\Temp и C:\Windows\Temp, удаляя файлы с расширениями: .tmp, .log, .cache, .temp, .old, .bak.

🔄 Удаление дубликатов
Использует MD5-хеширование для точного определения одинаковых файлов на рабочем столе пользователя.

🌐 Очистка кэша браузеров
Поддерживается очистка кэша Google Chrome и Microsoft Edge.

📊 Статистика дисков
Отображает информацию о всех доступных дисках: общий объём, занятое пространство, свободное место (в байтах, КБ, МБ, ГБ).
